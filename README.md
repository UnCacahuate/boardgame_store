# 🎲 Sorcerer Store

Ejercicio de maquetación responsive desarrollado para practicar CSS Grid y Flexbox.

## 🔗https://uncacahuate.github.io/boardgame_store/


## 🎯 Objetivo

Construir una tienda ficticia de juegos de mesa con diseño adaptable para dispositivos móviles, tablets y escritorio.

## 🛠️ Tecnologías

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Media Queries
- Font Awesome
